## Module 8: Data Lookup with Dictionaries & Basic Aggregation  
### GlobalTech Solutions Customer Management System  

### Files Included  
module8-assignment.py  

### What I Practiced  
In this assignment, I practiced using Python dictionaries to manage and organize business data. I worked with nested data structures to store customer and project information, performed lookups, and applied aggregation techniques. I also developed functions to analyze data, validate inputs, and generate insights, strengthening my ability to simulate real-world data systems.

### Key Concepts  
- Dictionary creation and nested data structures  
- Data lookup and retrieval using keys  
- Updating and modifying dictionary values  
- Iteration with loops for data processing  
- Aggregation (totals, averages, counts)  
- Dictionary comprehensions for efficient transformations  
- Function creation for validation and analysis  
- Basic recommendation system logic  

### Summary  
This project demonstrates how dictionaries can be used to build a simple customer management system. By organizing customer and project data, calculating financial metrics, and generating insights, the program simulates real-world business operations. It highlights how structured data and Python logic can support decision-making, service recommendations, and overall business analysis.
